# QuickLauncher 第一版开发任务清单

## 第一阶段：基线

1. 整理工程基础结构
   - verify: 程序可编译
2. 加入托盘服务
   - verify: 关闭主窗口后进程仍在，托盘菜单可操作
3. 加入全局热键服务
   - verify: 任意前台程序中可唤起主浮层
4. 完成主浮层 UI 骨架
   - verify: 输入框自动聚焦，`Esc` 可关闭
5. 完成设置页骨架
   - verify: 可打开、可保存、重启后配置仍在
6. 加入开机启动开关
   - verify: 设置变化能同步到当前用户启动项

## 第二阶段：索引

1. 扫描开始菜单 `.lnk`
2. 扫描桌面快捷方式
3. 枚举 `UWP`
4. 扫描自定义目录
5. 去重与噪音过滤
6. 写入本地索引缓存

verify:

- 可得到一批可启动应用
- 无明显重复

## 第三阶段：搜索

1. 名称归一化
2. 分词与别名字段生成
3. 拼音全拼生成
4. 拼音首字母生成
5. 评分器实现
6. 最近使用权重接入

verify:

- 中文、英文、拼音、首字母都可命中

## 第四阶段：启动与体验

1. `.exe` 启动
2. `.lnk` 启动
3. `UWP` 启动
4. 上下键选择
5. `Enter` 启动
6. 鼠标点击启动

verify:

- 搜索首项可成功打开
