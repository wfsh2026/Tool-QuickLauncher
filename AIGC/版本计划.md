# QuickLauncher 第一版版本计划

## 目标

第一版只解决一件事：像 iOS Spotlight 一样，用全局快捷键快速搜索并启动 Windows 应用。

## 范围

### 第一版纳入

- 全局快捷键唤起
- 搜索浮层主界面
- 托盘常驻
- 设置页
- 配置持久化
- 应用索引
- 中文、英文、拼音、模糊搜索
- 统一启动 `.lnk`、`.exe`、`UWP`
- 最近使用加权

### 第一版不纳入

- 全盘扫描 `.exe`
- 文件搜索
- 命令模式
- 插件系统
- 高级主题系统
- 实时文件监听增量索引

## 技术决策

- 语言：`C#`
- UI：`WPF`
- 运行时：优先建议 `.NET 8`，但当前开发机仅安装 `.NET 10 SDK`，本仓库现阶段保留 `net10.0-windows`
- 架构：轻量 `MVVM`
- 本地持久化：`JSON`
- Windows 集成：`Shell API / COM / UWP 枚举`

## 里程碑

### M1 基线

- 托盘
- 主浮层窗口
- 设置窗口
- 全局快捷键
- 配置读写

完成标准：

- 程序可启动
- 可最小化到托盘
- 可通过热键显示或隐藏主浮层
- 可打开设置并保存基础配置

### M2 索引

- 开始菜单扫描
- 桌面扫描
- UWP 枚举
- 自定义目录扫描
- 本地索引缓存

完成标准：

- 能生成一批真实可启动应用
- 二次启动优先加载缓存

### M3 搜索

- 名称前缀与子串匹配
- 拼音全拼匹配
- 拼音首字母匹配
- 简单模糊匹配
- 最近使用加权

完成标准：

- `wx` 可命中“微信”
- `weixin` 可命中“微信”
- `term` 可命中 `Windows Terminal`

### M4 启动与体验

- 统一启动逻辑
- 键盘导航
- 鼠标点击启动
- 空输入显示最近使用

完成标准：

- `Enter`、鼠标点击都能真实打开目标应用
- 关闭主窗口不退出
