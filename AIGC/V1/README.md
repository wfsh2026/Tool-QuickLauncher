- 全局快捷键唤起主搜索浮层
- 开始菜单、桌面、UWP、自定义目录的本地索引

- 上下键、Enter、鼠标单击启动应用
- 设置页修改快捷键、开机启动、启动后隐藏、搜索范围、自定义目录
- 系统托盘常驻，关闭主窗口不退出
